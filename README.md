## tmux
```shell
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
ln -s ~/GithubRepos/dotfiles/.tmux.conf .tmux.conf
tmux 
<Ctrl-b> + I (install plugins)
<Ctrl-b> + r (reload tmux)
```
